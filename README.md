# Sons of Eternity

Мультиплатформенная пошаговая РПГ с глобальной гексагональной картой и системой крафта.

## 🎮 Особенности игры

- **Мультиплатформенность**: Игра доступна на Android и PC (Windows, Linux, macOS)
- **Глобальная гексагональная карта**: Исследуйте огромный мир, разделённый на гексы
- **Пошаговые бои**: Тактические сражения на отдельных локациях
- **Система крафта**: Создавайте оружие, броню, зелья и другие предметы

## 🛠 Технологии

- **Язык программирования**: Java
- **Фреймворк**: [libGDX](https://libgdx.com/) - кроссплатформенный игровой движок

## 📋 Требования

### Для сборки и разработки
- JDK 8 или выше
- Gradle
- Android SDK (для сборки под Android)

### Для запуска
- **PC**: Windows 10/11, Linux или macOS
- **Android**: Android 5.0 (API level 21) или выше

## 🚀 Сборка и запуск

### Сборка для PC
```bash
./gradlew desktop:run
```

### Сборка для Android
```bash
./gradlew android:assembleDebug
```

APK файл будет создан в `android/build/outputs/apk/`

### Сборка всех версий
```bash
./gradlew build
```

## 🎯 Геймплей

### Глобальная карта
- Перемещение по гексагональной сетке
- Различные типы местности
- Случайные события при исследовании

### Боевая система
- Пошаговый режим боя
- Отдельные локации для сражений
- Тактическое позиционирование на гексах

### Крафт
- Сбор ресурсов во время исследования
- Создание предметов через меню крафта
- Рецепты открываются по ходу игры

## 📁 Структура проекта

```
├── core/           # Основной код игры (общий для всех платформ)
```
